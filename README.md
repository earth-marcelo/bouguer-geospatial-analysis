# Geospatial Data Analysis for Bouguer Anomaly Modeling

## Overview
This project applies data analysis techniques to integrate and process multiple geospatial datasets, generating a regional Bouguer anomaly model over the Rio Grande Rise.

The focus is on handling heterogeneous data sources, performing data preprocessing, and extracting meaningful spatial patterns from complex scientific datasets.

## Objectives
- Integrate multiple datasets with different resolutions and uncertainties  
- Apply data processing and transformation techniques  
- Perform spatial data analysis  
- Generate interpretable visual outputs  

## Data
The project combines:
- Topographic data  
- Free-air gravity anomaly  
- Sediment thickness  
- Crustal thickness  

## Methodology
- Data cleaning and preprocessing  
- Integration of heterogeneous datasets  
- Transformation and normalization  
- Spatial analysis and visualization  

## Tools
- GMT (Generic Mapping Tools)  
- Python (if applicable)

## Results
The final output is a Bouguer anomaly (MBA) map highlighting spatial variations across the study area.

## Key Takeaways
- Application of data analysis techniques to scientific datasets  
- Integration of heterogeneous geospatial data  
- Extraction of meaningful spatial patterns from complex data
